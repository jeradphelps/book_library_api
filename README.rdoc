
# Book Library Api

## To Run Locally

    > git clone https://github.com/jeradphelps/book_library_api.git
    > cd book_library_api
    > bundle
    > rake db:migrate
    > rails s
    > in your browser, visit http://localhost:3000

## Full API Endpoint Docs

    > rails s
    > In your browser, visit: http://localhost:3000/apidoc
